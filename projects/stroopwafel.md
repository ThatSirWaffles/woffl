A 1980s terminal. Created February 2024.

<i>
Presenting Stroopwafel, a chunky experimental 1980s terminal to perform very simple tasks. Includes a nice little API to interact with core machine functionality (displaying text, playing audio, interacting with cartridges and so on) and an expandable system for commands, where each command is just a modulescript which defines its name, format, description, and function to execute. These 2 systems combined make it really simple to add new functionality.<br/><br/>This version has a small extension module called Li√®ge (a type of Belgian waffle) to read and write to cartridges, which act as a means of storing data to use on Stroopwafel.
</i>

- "Hardware" APIs exposed to custom commands|
